Cyber Funk

A text based adventure game made in C++. This was my first project using the language.

Team Size: 1
Time Frame: 4 months
Language: C++

About

For my first C++ assignment during my first year at university I was challenged to make a text based adventure game in C++. I created a game called Cyber Funk, set in a cyberpunk city in the future, in which people fight each other through dance battles.

As I had already coded in other languages before this assignent, I wanted to create a game that was a bit more complex than a basic linear game. I created a system where I could create locations, place NPCs in them, and decide whether they were locked on not and what item was needed to unlock them. The player could then move through these locations in the game in whatever way they wanted, entering them in any order and having to search through them and fight the NPCs to earn the items that would unlock other locations.
