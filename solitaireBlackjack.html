Blackjack / Solitaire

A remake of the traditional card games made entirely in C++ during my first year of university.

Team Size: 1
Time Frame: 5 months
Language: C++

About

During my first year of university, whilst learning C++, I was given an assignment to create a C++ program that lets you play both Blackjack and Solitaire. The purpose of this project was to teach us about creating modular, reusable code, as both games would need things like a card class, a player class etc.

I made the blackjack game first, which follows the basic rules of the card game, including the ability to split the deck. I made sure to use functions and classes that I would be able to later reuse for the solitaire game.

Creating solitaire was quite challenging, as I made the decision to render the entire game using ASCII. I reused all of the classes and functions that I made for the blackjack game, and also added extra features such as arrow key controls and colours.
